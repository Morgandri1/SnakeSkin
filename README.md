# SnakeSkin
very simple, typewritter esc text editor.

requirements:
CopperUI ([GitHub](https://github.com/Copperhead-Software/CopperUI))

how to use:

python3 Snakeskin.py <filename>

if Snakeskin cannot find your file, it will create one for you.
